# 🛒 E-Commerce Website

A modern and dynamic Ecommerce website built using **HTML, CSS, and JavaScript**.  
This project demonstrates front-end web development skills including layout design, dynamic rendering, DOM manipulation, and interactive user features.

---

## 🚀 Features

- Clean and modern user interface
- Product listing section
- Dynamic shopping cart functionality
- Real-time price calculation
- Coupon system
- Tax and total calculation
- Remove product functionality
- Checkout interaction
- Organized layout with reusable structure

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- DOM Manipulation
- Basic Business Logic Implementation

---

## 📂 Project Structure

- `index.html` – Main structure of the website
- `style.css` – Styling and layout design
- `script.js` – Dynamic functionality and logic
